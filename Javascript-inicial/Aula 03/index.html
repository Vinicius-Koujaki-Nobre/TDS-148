HTML

HTML é uma linguagem de marcação de hipertexto que define a estrutura e o significado do conteúdo de uma página web. É o componente básico da web e é usado para construir websites.

JAVASCRIPT

JavaScript (JS) é uma linguagem de programação que permite adicionar interatividade a páginas da web. É a linguagem padrão interpretada pelos navegadores e é considerada uma das mais populares no mundo.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo que define como os elementos de uma página web aparecem. É utilizada para estilizar textos, cores, espaçamento, fontes, tabelas, imagens e layouts.

JAVA

Java é uma linguagem de programação orientada a objetos, multiplataforma e rápida. É uma das linguagens mais utilizadas por desenvolvedores.
